# Meal Planner

A phone-friendly meal planner. One file, no server, no accounts. Your data saves in your phone's browser.

## Put it online (GitHub Pages)
1. Create a new GitHub repo (e.g. `meal-planner`).
2. Upload `index.html` to it.
3. Repo **Settings → Pages** → Source: `Deploy from a branch` → Branch: `main` / `root` → Save.
4. Wait ~1 minute, then open the link GitHub gives you (like `https://yourname.github.io/meal-planner`).

## Add to your home screen
- **iPhone (Safari):** open the link → Share button → **Add to Home Screen**.
- **Android (Chrome):** open the link → ⋮ menu → **Add to Home screen / Install app**.

It opens full-screen like an app.

## Heads up on your data
Meals and grocery statuses are stored on **each device** (browser localStorage). It won't sync between your phone and computer, and clearing the browser's site data erases it. Plan on one main device.

## What it does
- 4 weeks of meals per plan; make separate plans (Summer, Winter, etc.) from the top-right menu.
- Each day has Breakfast / Lunch / Dinner. Tap a meal to add ingredients, sides, and a recipe (Pinterest/website link **or** cookbook + page number) plus notes.
- Add a day label like "Taco Tuesday" or "Try-it Thursday".
- **Grocery** tab builds a shopping list from that week's ingredients; tap **Out / Low / Got** on each item. "Got" crosses it off.
- **Grand List** tab does the same for the whole month.
- Items used in more than one meal show where they're used, so you can spot overlap.
